# Project Documentation Template

> Template for comprehensive project documentation structure.

## Structure

```
project-docs/
├── README.md       # This file - project overview
├── PLANNING.md     # Architecture and design decisions
├── TASK.md         # Task tracking and progress
└── KNOWLEDGE.md    # Learned patterns and solutions
```

## Files

### PLANNING.md
- Project architecture overview
- Technical stack decisions
- Feature roadmap
- Design patterns used

### TASK.md
- Current sprint tasks
- Completed work log
- Blockers and issues
- Priority queue

### KNOWLEDGE.md
- Lessons learned
- Reusable patterns discovered
- Error solutions for future reference
- Best practices established

## Usage

1. Copy this template to your project:
```bash
cp -r templates/project-docs ./docs
```

2. Customize each file for your project context

3. Keep documentation updated as you progress

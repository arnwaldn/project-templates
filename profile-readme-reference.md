## Quick Start: Create Your Profile README

```bash
# 1. Create a repo with your GitHub username
gh repo create YOUR_USERNAME --public --description "My GitHub profile"

# 2. Add a README.md
echo "# Hello! I'm [Your Name]" > README.md

# 3. Push it
git add README.md && git commit -m "feat: add profile README" && git push

# 4. Visit github.com/YOUR_USERNAME to see it live
```

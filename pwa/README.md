# Progressive Web App (PWA) Template

A production-ready Progressive Web App template built with Next.js 15, TailwindCSS 4, and next-pwa.

## Features

- **Installable** - Add to home screen on any device
- **Offline Support** - Service worker caching with Workbox
- **Push Notifications** - Native notification support
- **App-like Experience** - Standalone display mode
- **Auto Updates** - Automatic service worker updates
- **Responsive** - Works on all screen sizes

## Stack

- **Framework**: Next.js 15 (App Router)
- **PWA**: next-pwa (Workbox)
- **Styling**: TailwindCSS 4
- **Icons**: Lucide React
- **TypeScript**: 5.7

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Configuration

### manifest.json

Update `public/manifest.json` with your app details:

```json
{
  "name": "Your App Name",
  "short_name": "App",
  "description": "Your app description",
  "theme_color": "#000000",
  "background_color": "#000000"
}
```

### Icons

Add your app icons to `public/icons/`:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

### Service Worker

The service worker is automatically generated by next-pwa. Configure caching strategies in `next.config.ts`:

```typescript
export default withPWA({
  dest: "public",
  runtimeCaching: [
    // Add custom caching rules
  ],
})(nextConfig);
```

## Components

### InstallPrompt

Shows a prompt to install the app after 30 seconds:

```tsx
import { InstallPrompt } from "@/components/install-prompt";

// Add to layout.tsx
<InstallPrompt />
```

### OfflineIndicator

Shows a banner when the user is offline:

```tsx
import { OfflineIndicator } from "@/components/offline-indicator";

// Add to layout.tsx
<OfflineIndicator />
```

### usePWA Hook

Access PWA features in your components:

```tsx
import { usePWA } from "@/lib/use-pwa";

function MyComponent() {
  const {
    isInstalled,    // App is installed
    isOnline,       // Network status
    canInstall,     // Install prompt available
    installApp,     // Trigger install
    shareApp,       // Web Share API
    showNotification, // Push notification
  } = usePWA();

  return (
    <button onClick={installApp}>
      Install App
    </button>
  );
}
```

# Puzzle Game Template (2048)

A 2048 puzzle game with smooth animations and touch support.

## Stack

- **Phaser 3** - Game framework
- **TypeScript** - Type safety
- **Vite** - Dev server & bundler

## Getting Started

```bash
npm install
npm run dev
```

## Controls

- **Arrow Keys** - Slide tiles
- **Swipe** - Touch/mouse swipe
- **R** - Restart game

## Project Structure

```
src/
├── main.ts              # Entry point
├── scenes/
│   └── GameScene.ts     # Main game scene
├── grid/
│   ├── Grid.ts          # Grid rendering
│   └── Cell.ts          # Individual cell
└── mechanics/
    └── SlideLogic.ts    # 2048 game logic
```

## Features

- **Smooth Animations**: Tile sliding, merging, and spawning
- **Touch Support**: Swipe gestures for mobile
- **Score Tracking**: Local storage for best score
- **Responsive**: Scales to fit screen
- **Win/Lose States**: Reaching 2048 or no moves

## Customization

### Grid Size

Change in `GameScene.ts`:
```typescript
this.grid = new Grid(this, 4, 4, 90, 10)  // rows, cols, cellSize, padding
this.logic = new SlideLogic(4, 4)          // rows, cols
```

### Tile Colors

Edit `TILE_COLORS` in `Grid.ts`:
```typescript
const TILE_COLORS = {
  2: { bg: '#eee4da', text: '#776e65' },
  4: { bg: '#ede0c8', text: '#776e65' },
  // ...
}
```

### Match-3 Variant

To convert to Match-3, replace `SlideLogic` with `Match3Logic`:
- Grid of colored gems instead of numbers
- Swap adjacent tiles
- Match 3+ of same color
- Cascade and combo system

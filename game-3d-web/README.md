# 3D Web Game Template

A production-ready Three.js 3D game template with physics, audio, and modular architecture.

## Stack

- **Three.js r170+** - 3D graphics
- **Cannon-es** - Physics engine
- **Howler.js** - Audio system
- **TypeScript** - Type safety
- **Vite** - Fast dev server & bundler

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
src/
├── main.ts              # Entry point
├── Game.ts              # Main game class
├── entities/            # Game objects
│   ├── Player.ts
│   ├── Entity.ts
│   └── Camera.ts
├── systems/             # Core systems
│   ├── InputSystem.ts   # Keyboard/mouse handling
│   ├── PhysicsSystem.ts # Cannon-es integration
│   └── AudioSystem.ts   # Howler.js integration
└── utils/
    ├── AssetLoader.ts   # GLTF/texture loading
    └── Debug.ts         # FPS counter, stats
```

## Controls

- **WASD** - Move
- **Mouse** - Look around (click to capture)
- **Space** - Jump

## Features

- Modular entity-component architecture
- Cannon-es physics with ground collision
- Spatial audio support
- GLTF/GLB model loading with Draco compression
- Responsive design
- Debug panel (dev mode only)

## Building for Production

```bash
npm run build
npm run preview
```

## Customization

### Adding Models

```typescript
const loader = new AssetLoader()
await loader.loadModel('player', '/assets/player.glb')
const model = loader.cloneModel('player')
scene.add(model)
```

### Adding Sounds

```typescript
audioSystem.loadSound('jump', { src: '/assets/jump.mp3' })
audioSystem.playSound('jump')
```

### Physics Bodies

```typescript
physicsSystem.addBox(position, { x: 1, y: 1, z: 1 }, mass)
physicsSystem.addSphere(position, radius, mass)
```

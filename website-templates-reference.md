## How to Use

### Browse live demos
```
https://learning-zone.github.io/website-templates/
```

### Clone a specific template
```bash
# Clone full repo
git clone --depth 1 https://github.com/learning-zone/website-templates.git

# Copy a single template
cp -r website-templates/bestro-restaurant-bootstrap-html5-template ./my-project
```

### Convert to React/Next.js
1. Copy HTML structure as JSX
2. Replace class= with className=
3. Extract CSS into Tailwind or CSS modules
4. Split into components
5. Add routing and dynamic data

# Roguelike Game Template

A classic roguelike template with ECS architecture and procedural dungeon generation.

## Stack

- **Phaser 3** / Canvas - Rendering
- **rot.js** - Procedural generation, FOV, pathfinding
- **TypeScript** - Type safety
- **Vite** - Dev server & bundler

## Getting Started

```bash
npm install
npm run dev
```

## Controls

- **Arrow Keys / WASD / hjkl** - Move (vi-keys supported)
- **y/u/b/n** - Diagonal movement
- **.** or **Space** - Wait
- **>** - Descend stairs

## Features

- **ECS Architecture**: Clean entity-component-system design
- **Procedural Dungeons**: BSP-based room generation using rot.js
- **Field of View**: Shadowcasting for realistic visibility
- **A* Pathfinding**: Smart enemy movement
- **Turn-Based Combat**: Classic roguelike mechanics
- **Multiple Enemy Types**: Goblins, Orcs, Trolls
- **Dungeon Floors**: Descend deeper with stairs

## Customization

### Adding Enemies

```typescript
const enemy = world.createEntity()
world.addComponent(enemy, new Position(x, y))
world.addComponent(enemy, new Renderable('D', '#ff0000', 'Dragon'))
world.addComponent(enemy, new Fighter(200, 25, 10))
world.addComponent(enemy, new AI())
```

### Custom Map Generation

Modify `DungeonGenerator.ts` to use different algorithms:
- `RotMap.Cellular` - Cave-like maps
- `RotMap.Uniform` - Uniform room distribution
- `RotMap.Rogue` - Classic Rogue-style

# Multiplayer Game Template

Real-time multiplayer game template using Phaser 3 and Colyseus.

## Stack

- **Client**: Phaser 3, TypeScript, Vite
- **Server**: Colyseus, TypeScript, Node.js
- **Shared**: Common types and constants

## Getting Started

```bash
# Install all dependencies
npm install

# Start development (client + server)
npm run dev
```

Server runs on `ws://localhost:2567`
Client runs on `http://localhost:5173`

## Features

- Lobby system with ready check
- Real-time player synchronization
- Client-side prediction
- Server reconciliation
- Chat system
- Docker deployment ready

## Networking

### State Synchronization
The server maintains authoritative game state using Colyseus schemas. State changes are automatically synchronized to all connected clients.

### Client-Side Prediction
Local player inputs are applied immediately for responsive gameplay. The client reconciles with server state to correct any prediction errors.

### Message Types
- `playerInput` - Player movement/action input
- `chatMessage` - Chat messages between players
